# ExplorePRO_Travel
**BASIC DESCRIPTION**
Explore PRO is a simple website to help travelers known where to go! It allows the user to experience the 7 different continents and learn more about them. They can click on the navigation bar on the top and be able to select a continent. They can then view information about that continent, followed by the states/territories there! There is also a Goal section to look at our goals for this website and then world facts about the world in general! More features are coming soon!

**GOAL**
I mainly made this website so that, firstly, I can learn and improve my website designing skills, which I'm proud of because I definitely saw improvement from my first commit to last. Secondly, I also am really interested in traveling and learning about other cultures, so I also did this project with that in mind! This is supposed to be like a digital version of a board I would have in real life with the places I want to go! All the places mentioned in the continent pages are places I want to visit one day!

**TIME USAGE**
Most time and effort was put into the UI and for it to have an appealing look. Hopefully I did good because I'm just learning how to code in HTML and CSS now! I was looking a lot of W3Schools to find out how to add shadows, rounding, and more to the pictures! I also spent time on researching the places and then putting them down!

Some challenges I faced were with the UI. I also had a hard time knowing how I could make the photos on each continent page the same size, but later on I was able to make them about the same. I am a slow typer too so it took me longer. Knowing whether I could use in-line CSS in the HTML vs using the CSS file took me some time too.
